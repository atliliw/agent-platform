## 查看实时指标

SLO 指标会自动记录，可以通过以下方式查看：

```go
// 在代码中查询
status := mcpService.GetSLOStatus("tool_success_rate")
fmt.Printf("成功率: %.2f%%\n", status.Current*100)
fmt.Printf("Error Budget: %.2f%%\n", status.BudgetRemaining*100)
fmt.Printf("状态: %s\n", status.Status)
```
